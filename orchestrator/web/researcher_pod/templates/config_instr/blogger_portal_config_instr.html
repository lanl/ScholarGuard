For the pod to track events in Blogger, it must be registered with Google Cloud Console and the API key that Google assigns after registration must be saved in the pod. The following steps will help you through the process of registering the Pod with Hypothesis:

1. Login to Google

First sign into Google: Google login page.

2. Create an API Key

Upon registering, navigate to the Google Cloud console page and create an API key. Once generated, insert the API key in BOTH fields below.
