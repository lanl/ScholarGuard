For the pod to track events in Twitter, it must be registered with Twitter and the Client ID and Client Secret that Twitter assigns after registration must be saved in the pod. The following steps will help you through the process of registering the Pod with Twitter:

1. Register your pod with Twitter

Please visit the Twitter Apps Page and register your Pod as a Twitter App by clicking on the "Create New App" button on the top right corner of the page. The application details required to fill out the form to create the new app in Twitter is provided below:

  • Website : {{ url_for("index_page.on_index", _external=True) }}
  • Callback URL : {{ url_for("oauth1_portal.get_authorization", portal_name="twitter", _external=True)}}

2. Save Client ID and Client Secret

Upon successful registration of the app, click on the menu item that says "Keys and Access Tokens" in the resulting page. This page should show the API key and API Secret under the section called "Application Settings". Please copy and paste the API key in the field Client ID and API Secret in the field Client Secret in the form below and hit "Save".
