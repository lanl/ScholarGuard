For the pod to track events in SlideShare, it must be registered with SlideShare and the API Key and Secret that SlideShare assigns after registration must be saved in the pod. The following steps will help you through the process of registering the Pod with SlideShare:

1. Fill the Registration Form

Apply for an API key by filling out the form at the SlideShare API page.

2. Save API Key and Secret

Upon registering, SlideShare will send you the API Key and Secret in an email. Please copy and paste that information in the fields below.
