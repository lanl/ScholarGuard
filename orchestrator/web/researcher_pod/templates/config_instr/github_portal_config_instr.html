For the pod to track events in GitHub, the pod app must be registered with GitHub. After registration, the Client ID and Client Secret that GitHub assigns must be saved in the pod. The following steps will help you through the process of registering the Pod with GitHub:

1. Register your pod with GitHub

Please visit the GitHub Settings Page and register your Pod as a GitHub App as explained in this GitHub Documentation. The information required to fill out the App registration form is provided below:

  • Homepage URL: : {{ url_for("index_page.on_index", _external=True) }}
  • Authorization Callback URL : {{ url_for("oauth2_portal.get_authorization", portal_name="github", _external=True)}}

2. Save Client ID and Client Secret

Upon successful registration of the app, you should see a page which shows what the Client ID and Client Secret are. Please copy that information and paste it in the corresponding fields in the form below and hit "Save".
