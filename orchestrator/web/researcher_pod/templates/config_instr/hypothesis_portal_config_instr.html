For the pod to track events in Hypothesis, it must be registered with Hypothesis and the API key that Hypothesis assigns after registration must be saved in the pod. The following steps will help you through the process of registering the Pod with Hypothesis:

1. Register for Hypthesis

First register for Hypothesis website: Hypthosis signup page.

2. Create an API Key

Upon registering, navigate to their developer page: Hypthosis API page. Once there, generate an API key and insert the API key in BOTH fields below.
